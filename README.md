# PMC面试题

PMC/跟单/计划岗位面试刷题应用，参考驾考宝典交互设计，支持选择题和简答题两种题型。

## 功能

- **75道面试题**：覆盖通用、跟单、生产相关、物料相关、计划相关、采购相关6大分类
- **选择题**：点击选项即时反馈对错，附深度解析
- **简答题**：翻转卡片模式，关键词提示→完整答案→一句话总结
- **重要度评分**：1-5星标注，聚焦高频必考题
- **收藏功能**：标记重点题目，专门练习收藏题
- **艾宾浩斯复习**：自动追踪学习记录，提醒待复习题目
- **分类筛选**：按岗位类别筛选题目
- **练习模式**：顺序练习、随机练习
- **PWA支持**：可添加到手机主屏幕，离线使用

## 运行方式

### 方式一：本地直接打开

双击 `index.html` 即可在浏览器中使用。

### 方式二：本地服务器

```bash
cd 面试宝典
python -m http.server 8080
```

浏览器访问 `http://localhost:8080`

### 方式三：GitHub Pages

推送至GitHub后，在仓库 Settings → Pages → Source 选择 main 分支即可在线访问。

## 技术栈

- Vue2（CDN引入）
- 单文件HTML应用，无需构建
- PWA（manifest.json + Service Worker）
